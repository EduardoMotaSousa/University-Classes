# University-Classes — C++ 🎓

![Status](https://img.shields.io/badge/status-em%20desenvolvimento-blue)

Repositório com exercícios e atividades práticas desenvolvidas durante o curso de Análise e Desenvolvimento de Sistemas (ADS).

O foco principal é praticar lógica de programação e fundamentos de C++, ao mesmo tempo em que aplico organização de projetos com Git e GitHub.

---

## 🌐 Interface Web

Este repositório possui uma interface web interativa que permite:

- Navegar pelas pastas do projeto
- Visualizar arquivos diretamente no navegador
- Acessar códigos sem precisar abrir o GitHub
- Ler este README integrado ao site

🔗 Acesse:  
https://eduardomotasousa.github.io/University-Classes/

## 🧠 Objetivo do Projeto

- Organizar os estudos de forma visual
- Praticar versionamento com Git
- Entender como consumir APIs (GitHub)
- Ter um primeiro contato com desenvolvimento web

---

## 🤖 Uso de Inteligência Artificial

Parte da interface web (HTML, CSS e JavaScript) foi desenvolvida com auxílio de IA.

O objetivo foi:
- Aprender na prática como funciona o front-end
- Entender integração com APIs
- Ajustar e evoluir o código manualmente

## ⚙️ Funcionamento

O site consome dados diretamente da API do GitHub:

- Lista pastas e arquivos dinamicamente
- Atualiza automaticamente ao adicionar novos arquivos
- Não precisa de backend
