Exercícios e atividades práticas desenvolvidas durante as aulas de
programação no curso de **Análise e Desenvolvimento de Sistemas (ADS)**
da Estácio. Foco em lógica de programação e fundamentos de C++.

---

## 🌐 Interface Web

Este repositório conta com uma interface web interativa que permite
navegar pelas pastas, visualizar arquivos diretamente no navegador
e explorar a estrutura do projeto de forma dinâmica.

🔗 **[Acesse aqui](https://eduardomotasousa.github.io/University-Classes/)**

---

## 🤖 Uso de Inteligência Artificial

Parte da interface web (HTML, CSS e JavaScript) foi desenvolvida com
auxílio de IA, com o objetivo de aprender conceitos de desenvolvimento
web, entender integração com APIs do GitHub e explorar boas práticas
de organização de projetos. Todo o código foi estudado e compreendido
durante o processo.
